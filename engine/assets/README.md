# Assets

## Purpose
- Engine source tree.
- Asset management and import pipelines.

### Subdirectories
- `include/` – Public headers.
- `samples/` – Sample content.
- `shaders/` – Shader code.
- `src/` – Source files.
- `tests/` – Automated test suites.

### Files
- `.gitkeep` – Placeholder to keep the directory in git.
- `CMakeLists.txt` – CMake build configuration.
